# Shared Assets

- `controls/` – accent-tinted UI glyphs (checkboxes, radios, tree toggles,
  toolbar overflow, splitter handles, dock titlebar buttons) referenced by the
  stylesheet as `:/uitheme/common/controls/<name>.svg`.

The build passes this folder to `make-resource.py` as `-include-dir`, which
embeds it under the `common/` alias prefix — that is what makes those
`:/uitheme/common/...` paths resolve. Every file here must be referenced by the
stylesheet; unreferenced ones are not packaged, and a reference with no file
fails the build.

Most of these are generated by `src/nova-dark/scripts/download_phosphor_icons.py`.

This folder used to also hold `styles/BaseTheme.scss`, shared with the old
Mumble Dark theme. With one theme left, those rules now live in
`src/nova-dark/source/_widgets.scss`.
